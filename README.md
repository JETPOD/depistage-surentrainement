# Dépistage du surentraînement — NutriCellScience

Outil d'auto-évaluation du risque de surcharge et de surentraînement (overreaching / OTS),
fondé sur la littérature scientifique (consensus IOC/ECSS-ACSM).

Application web autonome (un seul `index.html`, 100 % côté client), bilingue FR/EN,
avec double niveau de lecture (grand public / professionnel).

> ⚠️ Outil d'orientation et de sensibilisation, **pas un diagnostic médical**.
> Le surentraînement est un diagnostic d'exclusion qui relève du médecin.

## Méthodologie

Score multidimensionnel pondéré :
- Charge d'entraînement (sRPE Foster, ACWR) — 20 %
- État perçu (indice de Hooper, dimensions POMS/RESTQ-Sport/MTDS) — 40 %
- Marqueurs objectifs (HRV, FC repos, performance, poids) — 30 %
- Drapeaux rouges médicaux (Meeusen 2013, IOC REDs 2023) — override

NutriCellScience — Mark DOWN
